Canvas

The canvas element provides a drawing surface for rendering graphics, animations, charts, and other visualizations dynamically within a web page using JavaScript.

Video

The video element allows embedding video content directly into web pages without requiring third-party plugins like Adobe Flash or Silverlight. Developers can specify multiple video sources (e.g., different formats or resolutions) using the source element within video, ensuring cross-browser compatibility and fallback options.

Audio

the audio element enables embedding audio files (e.g., music, podcasts, sound effects) directly into web pages. Developers can include multiple audio sources using source elements to ensure compatibility across different browsers and platforms.
